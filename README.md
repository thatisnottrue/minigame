# 생태 협동학습 궤적 미니게임

HTML/CSS/JavaScript만으로 만든 가상·현실 연계 생태 협동학습 MVP입니다. 3인 1조 학생이 각자의 전문가 패스코드로 입장해 공통 3단계 스테이지를 완수하고, 역할별 글자와 함정 해제 단서를 조합해 최종 보물 장소인 **교무실**을 도출합니다.

## 게임 구조

1. **패스코드 입력**
   - `1004`: 식물·곤충 전문가 — 최종 글자 `교`
   - `2005`: 미생물·어류 전문가 — 최종 글자 `무`
   - `3006`: 조류·포유류 전문가 — 최종 글자 `실 / 방` 동시 노출
2. **STAGE 1 - 먹이그물 잇기**
   - 경포가시연습지 실제 생물 카드(가시연, 연테두리진딧물, 송사리, 수달, 미생물 등)를 클릭해 `먹히는 생물 ➜ 먹는 생물` 방향의 다중 먹이그물과 분해자 순환 연결을 모두 완성합니다.
3. **STAGE 2 - 인과관계 궤적**
   - 2D 탑뷰 그리드에서 방향키/WASD 또는 모바일 가상 패드로 이동합니다.
   - 역할별 정답 인과관계 노드를 순서대로 밟아 Trace 선을 완성합니다.
   - 오답·교란 노드 충돌 시 경고가 뜨고 현재 스테이지가 출발점으로 리셋됩니다.
4. **STAGE 3 - 생태 피라미드**
   - 생산자부터 최상위 소비자까지 에너지 10% 법칙에 맞춰 블록을 쌓습니다.
5. **결과 화면**
   - 역할별 글자와 최종 함정 해제 힌트를 표시합니다.
   - `localStorage`에 역할별 클리어 여부와 최고 클리어 타임을 저장합니다.

## 기술 조건

- 정적 파일 3개(`index.html`, `style.css`, `script.js`)로 구성
- 외부 유료 라이브러리 없음
- Netlify 배포 시 Build command 없이 Publish directory `.`로 설정 가능
- 태블릿/모바일 대응 반응형 레이아웃 및 가상 이동 패드 제공

## 실행 방법

별도 빌드 없이 정적 파일로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000`을 열면 됩니다.
