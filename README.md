# 메타포트 생태계 인과관계 미니게임

이 저장소는 HTML 기반 미니게임 프로젝트입니다.

## 게임 기본 골자

### 게임 장르

2D 탑뷰 그리드 인과관계 추론 퍼즐 게임

### 플레이 방식

역할별 패스코드를 입력해 진입한 뒤, 2D 바둑판 화면에서 방향키로 캐릭터를 조작하여 활동지 속 생태계 인과관계 순서에 맞는 단어 좌표들을 순서대로 밟아 궤적(Trace) 선을 연결합니다.

### 게임 목표

오답 노드를 피해 생태계 연쇄 반응 노드들을 올바르게 순서대로 밟아 3단계를 모두 클리어하고, 화면에 완성된 궤적 글자(교/무/실)와 역할별 최종 해제 힌트를 획득합니다.

### 게임오버 조건

잘못된 인과관계 단어(오답 노드)와 충돌 시 시스템 경고와 함께 현재 스테이지의 진행 상황이 초기화되며 출발점으로 리셋됩니다.

### 점수 시스템

별도의 스코어 시스템 대신 스테이지별 정답 노드 순차 밟기 성공 여부(체크포인트 방식)로 진행되며, 오답 충돌 시 현재 스테이지 궤적 리셋 페널티가 부여됩니다. 최고 정답 연결 수는 `localStorage`에 저장됩니다.

### 화면 구성

- 패스코드 입력(시작) 화면
- 2D 탑뷰 미로 게임 화면 (Stage 1~3)
- 궤적 글자 완성 및 역할별 힌트 팝업(결과) 화면

## 기술 조건

- HTML, CSS, JavaScript만 사용
- `index.html`, `style.css`, `script.js` 파일로 분리
- 외부 유료 라이브러리 사용 금지
- Netlify에서 Build command 없이 배포 가능
- Publish directory는 `.`로 설정
- 모바일에서도 플레이 가능하도록 반응형 구성
- `localStorage`를 사용해서 최고 정답 연결 수 저장

## 실행 방법

별도 빌드 없이 정적 파일로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000`을 열면 됩니다.

## 테스트용 패스코드

- `ECO-TEACHER`: 교 역할
- `ECO-FIELD`: 무 역할
- `ECO-LAB`: 실 역할
