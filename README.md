# 생태 협동학습 미니게임

HTML/CSS/JavaScript만으로 만든 가상·현실 연계 생태 협동학습 MVP입니다. 3인 1조 학생이 각자의 전문가 패스코드로 입장해 공통 3단계 스테이지를 완수하고, 역할별 글자를 조합해 최종 보물 장소인 **과학실**을 도출합니다.

## 게임 구조

1. **패스코드 입력**
   - `1004`: 식물·곤충 전문가 — 최종 글자 `과`
   - `2005`: 미생물·어류 전문가 — 최종 글자 `학`
   - `3006`: 조류·포유류 전문가 — 최종 글자 `실`
2. **STAGE 1 - 먹이그물 잇기**
   - 경포가시연습지 실제 생물 카드(가시연, 연테두리진딧물, 송사리, 수달, 미생물 등)를 클릭해 `먹히는 생물 ➜ 먹는 생물` 방향의 다중 먹이그물과 분해자 순환 연결을 모두 완성합니다.
3. **STAGE 2 - 경포습지 생태계 통합 GROW 복원 게임**
   - 세 학생 모두 동일한 GROW 복원 시스템을 플레이합니다.
   - 하단의 6개 생태 분류군 버튼을 `미생물 ➜ 가시연(식물) ➜ 수서곤충 ➜ 어류 ➜ 조류 ➜ 포유류` 순서로 클릭합니다.
   - 올바른 순서일 때만 기존 생물이 레벨업하고, 오답 선택 시 “생태계 평형이 맞지 않아 천이가 중단되었습니다!” 알림과 함께 캔버스가 초기화됩니다.
   - 완벽 복원 시 접속 역할에 따라 `과`, `학`, `실` 중 하나의 힌트 글자가 중앙에 표시되고 STAGE 3 진입 버튼이 활성화됩니다.
4. **STAGE 3 - 생태 피라미드 밸런스 퍼즐**
   - 위기 상태에서는 1차 소비자 층만 비정상적으로 넓어진 3단 생태 피라미드를 보여줍니다.
   - `생산자 감소 & 2차 소비자 증가` ➜ `1차 소비자 감소` ➜ `생산자 증가 & 2차 소비자 감소 (평형 회복)` 순서로 인과관계 카드를 드래그 앤 드롭해 배열합니다.
   - 정답이면 Step 1 ➜ Step 2 ➜ Step 3 애니메이션으로 피라미드가 안정적인 정삼각형 비율로 회복됩니다.
5. **결과 화면**
   - 모든 방탈출 미션 성공 엔딩과 역할별 글자, 최종 장소 조합 힌트를 표시합니다.

## 기술 조건

- 정적 파일 3개(`index.html`, `style.css`, `script.js`)로 구성
- 외부 유료 라이브러리 없음
- Netlify 배포 시 Build command 없이 Publish directory `.`로 설정 가능
- 태블릿/모바일 대응 반응형 레이아웃 제공

## 실행 방법

별도 빌드 없이 정적 파일로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000`을 열면 됩니다.
